# Dual Clock WinForms

Простые часы на C# Windows Forms с двумя одинаковыми индикаторами времени.

<img width="548" height="202" alt="Снимок экрана 2026-03-01 181743" src="https://github.com/user-attachments/assets/5d37162e-1854-467c-8242-902833886652" />

## Функциональность

- Отображение текущего времени в двух label
- Формат времени: ЧЧ:ММ:СС
- Обновление каждую секунду через Timer

## Интерфейс

- label1 — первые часы
- label2 — вторые часы
- Timer с интервалом 1000 мс

## Куда можно встроить

- В демонстрационные приложения
- В виджеты для рабочего стола
- В учебные проекты по Timer

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Оба label показывают одинаковое время.
